## About Sumanto

Sumanto is a web-based system designed to streamline and simplify management and transaction processes.

## Installation Guide

- composer install
- cp .env.example .env
- php artisan storage:link
- php artisan key:generate
- php artisan migrate:fresh --seed

In one line cmd :

`composer install && cp .env.example .env && php artisan storage:link && php artisan key:generate && php artisan migrate:fresh --seed`

Additional command :
- npm install
- npm run dev, or
- npm run build
